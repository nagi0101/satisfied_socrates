h1~h6

제목은 h1부터 h6 태그를 통해, 큰 제목부터 작은 제목 순으로 표현할 수 있다.

목록

목록을 나타낼 땐 ul이나 ol 안에 li를 넣어 표시할 수 있다!

ul HTML <ul> 요소는 정렬되지 않은 목록을 나타냅니다. 보통 불릿으로 표현합니다.
ol HTML <ol> 요소는 정렬된 목록을 나타냅니다. 보통 숫자 목록으로 표현합니다.
li HTML <li> 요소는 목록의 항목을 나타냅니다. 반드시 정렬 목록(<ol>), 비정렬 목록(<ul>, 혹은 메뉴(<menu>) 안에 위치해야 합니다. 메뉴와 비정렬 목록에서는 보통 불릿으로 항목을 나타내고, 정렬 목록에서는 숫자나 문자를 사용한 오름차순 카운터로 나타냅니다.

a

a 태그를 사용할 때는 href attribute를 통해 무슨 페이지를 열지 지정해야한다. target attribute를 통해 페이지를 현재 페이지에서 열지 새로운 페이지에서 열지도 설정할 수 있으니 참고하자!

  • 새 창에서 열기
  • 현재 창에서 열기

img

img 태그를 사용할 때는 항상 ref attribute를 통해 이미지 파일의 경로를 지정해주어야 한다. url 또는 상대경로/절대경로를 통해 로컬에 있는 파일을 지정하자!

form

form과 input 태그를 통해 사용자들로부터 여러가지 입력을 받을 수 있다. 여러가지 input들을 시도해보자!
